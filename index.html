The Smell Engine

The Smell Engine is a software/hardware framework and system that facilitates the synthesis of physical odors in real-time, simulating what a user would experience in a virtual environment infused with smells. This is achieved through the Smell Composer design framework that integrates with the Unity Game Engine, allowing developers to program and configure odor sources. The system's Smell Mixer and Smell Controller components estimate the odor mix and coordinate an olfactometer to present the synthesized smell to the user. Tested through a series of PID system studies and user studies, the Smell Engine demonstrated the capability to produce odors at granularities finer than human olfactory detection sensitivity and significantly improved users' ability to accurately localize odors.

Mixed Reality Memories

This project is an amalgamation of technology and creativity, using RGBD sensors to capture volumetric footage of real-world events. The captured footage is then anchored to the location of the event using Azure Anchors, which enables users to experience the event in its original context. In addition, the project offers the flexibility to replay the event in any environment. The backend of the system uses a database to map Azure Anchors to the volumetric capture files, which are then loaded onto a device for playback. This approach allows for a unique and immersive way to relive events, bridging the gap between digital and physical space.

AI NPC

ChatbotAvatarAI is a dynamic Unity project that merges state-of-the-art technologies like OpenAI API, Azure Voice API, Google Cloud Speech to Text, and Oculus Lip Sync to craft an interactive AI-based chatbot interface. This framework facilitates seamless integration with YOLO-NAS, allowing the NPC to stream virtual camera frames and receive responses from a YOLO-NAS server instance, creating an immersive experience. Users can engage in a conversation with the NPC, receiving near-instantaneous responses in an Azure-generated voice, while the NPC can also be configured to visually perceive its environment with YOLO-NAS.​
